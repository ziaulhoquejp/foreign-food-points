# Foreign Food Points System

A full-stack membership and loyalty system built with **Next.js** and **Supabase**.

---

## 🚀 Features

- Member management system
- QR-based member identification
- Points system (+10 per action)
- Automatic coupon issuance (500 points rule)
- Coupon system (active / used tracking)
- Admin dashboard
- Member detail page with visit history
- Row Level Security (RLS) support

---

## 🧱 Tech Stack

- Next.js (App Router)
- React
- Supabase (Database + Auth + RLS)
- TypeScript
- Vercel (Deployment)

---

## 🎯 Business Logic

- Every action adds +10 points
- When points reach 500 → auto coupon is created
- Coupons can be tracked as active/used
- Admin can manage all members
